# Customer-behavior-orders
Data Analyst project (begineer)
This project focuses on analyzing customer ordering behavior and purchase patterns using structured sales and order data. The goal is to extract actionable insights related to customer total sales, order trends,avg rating products to support data-driven business decisions.

Using MySQL, the project explores how customers interact with products, identifies high-performing items, and analyzes ordering behavior across different dimensions such as time, gender, and ratings.

Tools & Technologies
MySQL
SQL Queries (JOINs, GROUP BY, CASE, Subqueries Functions)
power bi 
basic DAX fuction to creating measures,calculated columns and dashboards
